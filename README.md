# CALCULATOR  
    A simple calculator application that performs basic arithmetic operations.

# Features
    Addition (+)
   
    Subtraction (−)

    Multiplication (×)
 
    Division (÷)
    


# Example operations
    5 + 3 = 8
    10 - 4 = 6
    6 × 7 = 42
    20 ÷ 4 = 5
